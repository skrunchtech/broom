🔍

Three-pass scanning

Size filter → quick 4KB hash → full SHA256. Only hashes what it needs to, fast even on multi-TB drives.

🛡️

Safe by default

Archive mode moves files to a timestamped folder with a full manifest. Restore anything with one command.

⚡

Parallel hashing

Saturates your SSD using all CPU cores. No tuning required.

🖥️

Beautiful TUI

Built with Bubbletea. Navigate groups, mark files, preview actions — all from the terminal.

📜

Scriptable

Flags for CI/automation: --dry-run, --archive, --delete --yes. Pipe-friendly output.

🔗

Hardlink aware

Detects and skips hardlinks so you never accidentally count the same inode twice.

Install

# Homebrew (macOS / Linux) brew install skrunchtech/tap/broom
# Go install go install github.com/skrunchtech/broom/cmd/broom@latest
# Or download a binary from GitHub Releases github.com/skrunchtech/broom/releases

Usage

# Interactive TUI broom ~/Documents ~/Downloads
# Dry run — see what would be removed broom --dry-run /Volumes/Extreme\ SSD
# Archive duplicates for verification broom --archive=/tmp/broom-archive /Volumes/Extreme\ SSD
# Delete with confirmation broom --delete --keep=newest ~/backups
